# randomquotes
echo a random quote every time you start the terminal

![image-20220826133348725](README.assets/image-20220826133348725.png)

## How to use
`cp ./randomquotes.sh ~/.randomquotes`

Add `source ~/.randomquotes` to the end of ~/.bashrc
